git status

( မိမိ local စက်ထဲမှ file အား git နှင့် ချိတ်ဆက်ထားခြင်း ရှိ / မရှိ စစ်ဆေးခြင်း )


git add .

( မိမိ local စက်ထဲမှ folder နှင့် file များကို .git folder ထဲသို့ထည့်ခြင်း )


git commit -m "any text message"

( git ထဲမှာ မိမိ file နှင့် folder အား ဘာအတွက် ဘာဆိုတာ ​ဖော်ပြလိုသော message အတွက်သုံးခြင်း )


git push -u origin master/main

( git acc နှင့် vs code ချိတ်ဆက်ခြင်း။ login ဝင်ထားတဲ့ browser နဲ့ connect လုပ်ရမည် )
